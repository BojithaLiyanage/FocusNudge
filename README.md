# FocusNudge 💧

FocusNudge is a lightweight, unintrusive macOS menu bar application designed to help you build healthy habits. Whether it's taking a moment to hydrate, resting your eyes from the screen, or standing up to stretch, FocusNudge provides fully customizable screen overlays to gently remind you to take care of yourself throughout your workday.

## ✨ Features

- **Menu Bar Integration:** Runs quietly in your macOS menu bar without cluttering your Dock.
- **Custom Reminders:** Set a personalized message (e.g., "💧 Time to hydrate!").
- **Flexible Intervals:** Choose exactly how often you want to be reminded.
- **Active Hours:** Set your active schedule so FocusNudge only nudges you during your working hours, avoiding interruptions while you rest.
- **Customizable Appearance:** Control the background color and opacity of the reminder overlay to suit your style.
- **Smooth Animations:** Choose between Fade, Scale, and Slide animations for the overlay presentation.
- **Sound Alerts:** Optionally play a notification sound when a reminder triggers (includes built-in macOS sounds).
- **Preview Mode:** Test your settings instantly with the "Preview Now" button.

## 🛠️ Requirements

- macOS 13.0+ (or as configured in your project)
- Xcode 14.0+ (for building from source)
- Swift 5.0+

## 🚀 Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/BojithaLiyanage/FocusNudge.git
   ```
2. Open `FocusNudge.xcodeproj` in Xcode.
3. Build and run the project (`Cmd + R`).
4. Look for the water drop icon in your Mac's menu bar!

## ⚙️ Usage

- Click the **FocusNudge** (water drop) menu bar icon to reveal quick controls.
- Click **Start/Stop** to toggle the active reminder timer.
- In the **Preferences** window, adjust your active hours, overlay animations, sound, message, and the reminder interval.
